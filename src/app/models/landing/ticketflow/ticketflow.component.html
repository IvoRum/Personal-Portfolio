Ticket Flow

Spring boot Java Script PosgreSQL
From: 01.March.2023 To: 14.June.2023

Tickt Flow is an all-in-one solution for seamless ticketing and queue management and reservations. Written as a batch, this is Ticket flow constructed a MVC application. Beeing is constructed with a spring boot as the back end, with Posgresql as the database and Vanila JavaScript for the front end.
